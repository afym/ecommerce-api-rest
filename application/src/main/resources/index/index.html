E-commerce (books online) API REST

This is a simple e-commerce API REST base on Spark micro framework for Java and a simple IO file persistence. just for development Is a prototype 7h

author : angel.fym

General responses

You will get the following responses in all the endpoints

Item Description Value or sample
Invalid If your request body is malformed HTTP 400 status 
                            {
                                "message": "Your request is invalid. Please check the documentation",
                                "status": 400,
                                "error": true
                            }
Unauthorized If your request has not a valid token HTTP 401 status 
                            {
                                "message": "You are unauthorized for this resource",
                                "status": 401,
                                "error": true
                            }

Login service

This entry point receives your email and password and generates an access token.

Request sample :

Item Description Value or sample
Endpoint Invoke with POST method http://localhost:4567/authorization/login
Header Required headers Content-Type : application/json
Body Request body 
{"email" : "peter1223@gmail.com", "password" : "password"}
Body Request body 
{"email" : "vania112@outlook.com", "password" : "password"}

Response :

Item Description Value or sample
Body Response body 
                            {
                                "message": "ok",
                                "status": 200,
                                "error": true,
                                "data": {
                                    "token": "9G6QW6N5LEK5KTWDMX"
                                }
                            }
